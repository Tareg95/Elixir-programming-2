# 💻 Programming 2 – Problem Solving & Recursion (KTH)

## 📌 Overview
This repository contains solutions to assignments from the **Programming 2 course at KTH**.

The course focuses on developing problem-solving skills in C, including:
- Recursion
- Structured programming
- Basic data structures
- Algorithmic thinking

Each folder represents a separate assignment.

---

## 📁 Contents

- Advent of Code Day 1  
- Derivatives  
- Evaluate an Expression  
- List and Tree  
- Huffman  
- Monte Carlo  
- Morse Code  
- Tower of Hanoi  
- Train Shunting

## 🧠 Concepts Used

While this is a Programming 2 course, the following concepts are applied:

- Recursion (Tower of Hanoi)
- Trees (Huffman coding, Morse code)
- Stacks & Queues (Expression evaluation, Train shunting)
- Parsing techniques
- Simulation (Monte Carlo)
